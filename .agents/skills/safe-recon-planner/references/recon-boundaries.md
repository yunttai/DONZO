# Recon Boundaries

Fast profile:

- scope validation
- passive subdomain discovery
- DNS validation
- HTTP probing
- crawling
- safe scanner templates only
- normalize, dedupe, rank, report

Normal profile adds archive URLs, passive amass, selected port discovery, API doc
discovery, GraphQL discovery, parameter extraction, candidates, and evidence.

Deep profile adds asset expansion, JS analysis, source maps, safe content/API
route discovery, secret candidate extraction, candidate engines, advanced
ranking, and optional scanners. Deep mode still does not allow automatic
exploitation, secret use, takeover claim, brute force, DoS, or destructive tests.
