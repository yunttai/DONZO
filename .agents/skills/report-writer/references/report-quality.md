# Report Quality

A useful report is concise, scoped, and reproducible.

Do:

- Link to request/response or artifact paths.
- Explain why the target matters.
- Explain what remains unverified.
- Use neutral language for candidates.
- Include safe manual verification steps.

Do not:

- Paste secrets.
- Claim account takeover or data exposure without evidence.
- Include destructive reproduction steps.
- Hide scope uncertainty.
